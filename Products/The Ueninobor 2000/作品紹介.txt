//----------------------------------------------------------------------
// プロダクト概要説明
//----------------------------------------------------------------------
　動物ボールを上に移動させるごとにスコアが増えていきます。タップで発生する衝撃波でボールを操り、制限時間内に高スコアを目指すゲームです。
　道中のコインを取得したり、ブロックを破壊したりするとフィーバーゲージがたまり、ブロックを一撃で破壊できるようになります。
　
//----------------------------------------------------------------------
// プロダクト作成のきっかけ
//----------------------------------------------------------------------
　Line株式会社様が主催の「GAME BOOT CAMP」という、30時間でゲームを一本つくるイベントにおいて作成したゲームです。
　
//----------------------------------------------------------------------
// 制作期間
//----------------------------------------------------------------------
30時間


//----------------------------------------------------------------------
// 制作人数
//----------------------------------------------------------------------
　5人
　ゲーム部分とキャラクター選択のプログラムを担当しました。

　
//----------------------------------------------------------------------
// アピールポイント
//----------------------------------------------------------------------
　30時間という限られた時間の中で、このクオリティのゲームを作り上げたという部分です。
　また、効率よく作業をするためにバージョン管理ツールを使用することで、チームでの開発も非常にスムーズに行うことが出来ました。
