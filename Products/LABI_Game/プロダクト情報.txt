//----------------------------------------------------------------------
// プロダクト概要説明
//----------------------------------------------------------------------
　去年、ヤマダゲームセンターブラウザーゲームアプリへ応募させていただいた作品です。このコンテストの存在を知った時点で締め切りが迫っていたので、JavaScript＋enchant.jsを使用し、非常に短期間で作成しました。
　落ちてくる商品を、プレイヤーキャラであるらびたんを操作してゲットしようという2Dゲームです。ゲットした商品は、画面右の回収ボックスへ運んでいくこと下ろせます。詳しくはタイトル画面のヘルプをご覧ください。
　
　
//----------------------------------------------------------------------
// プロダクト作成のきっかけ
//----------------------------------------------------------------------
　ヤマダ電機グループのマスコットキャラクターであるらびたんとつくもたんが好きであったのと、なにかゲームを作って応募してみたいという時期に、校内で応募のチラシが張ってあるのを見て、制作をはじめました。
　
//----------------------------------------------------------------------
// 制作期間
//----------------------------------------------------------------------
2014年7月初めより
　3週間程度


//----------------------------------------------------------------------
// 制作人数
//----------------------------------------------------------------------
　1人

　
//----------------------------------------------------------------------
// 技術的に工夫した点、苦労した点
//----------------------------------------------------------------------
　技術的にかといわれると非常にあやしいのですが、初めて使用する言語とエンジンだったので、全て手探りでコードを書いていかなければならなかったのが非常に大変でした。自身にとって初めてのスクリプト言語であったこともあり、C、C++との違いに戸惑いながらの作業でした。
　また、デザイン関係の内容で、技術的ではないのですが、ゲーム内で使用されているドット絵を用意するのも大変でした。これも初めての経験でした。
