//----------------------------------------------------------------------
// プロダクト概要説明
//----------------------------------------------------------------------
　このゲームは、横スクロール避けゲーです。自機である光の玉をタッチによって操作し、闇の渦を避けながらひたすら右へ進みスコアを稼ぐというのが主な目的です。詳しくは付属の説明書をご覧ください。
　
　
//----------------------------------------------------------------------
// プロダクト作成のきっかけ
//----------------------------------------------------------------------
　このゲームは、日本ゲーム大正2014年　アマチュア部門への応募を目的として制作しました。
　
//----------------------------------------------------------------------
// 制作期間
//----------------------------------------------------------------------
2014年4月〜6月末、その後もこまめにブラッシュアップを重ねる
　3ヶ月程度


//----------------------------------------------------------------------
// 制作人数
//----------------------------------------------------------------------
　1人

　
//----------------------------------------------------------------------
// アピールポイント
//----------------------------------------------------------------------
　個人賞も狙うため個人制作が前提でした。その時点で、一番時間がかかるのはゲームバランスの調整だと考え、真っ先にそこの対策をしました。このゲームでのバランス調整というのは、障害物の配置が主となってきます。なので、配置位置を記録したテキストファイルを実行時に読み込みをすることによって、リビルドをせずとも障害物の配置、動作、個数を細かく設定できるようにしました。また、それらのデータもExcel上で管理し、指定範囲のデータをCSV形式で出力するようマクロを組みました。
　
　障害物オブジェクト生成に関しては、ファクトリパターンを使用しております。
　
　また、デザイナーの力を借りることができないので、丸や四角などの単純な図形を使用したパーティクルを用いることで、デザインのカバーをしています。
